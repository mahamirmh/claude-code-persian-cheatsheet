# چیت‌شیت دستورهای کاربردی Claude Code

این فایل یک مرجع سریع فارسی برای دستورهای پرکاربرد Claude Code است.

> نکته: همه دستورها ممکن است برای همه کاربران یا همه نسخه‌ها فعال نباشند. دستور دقیق محیط خودت را با `/help` بررسی کن.

---

## 1. دستورهای اصلی CLI

| دستور | کاربرد |
|---|---|
| `claude` | ورود به محیط تعاملی Claude Code |
| `claude "task"` | شروع Claude Code با یک دستور اولیه |
| `claude -p "query"` | اجرای یک درخواست و خروج از برنامه |
| `cat file.txt \| claude -p "explain"` | ارسال محتوای فایل به Claude |
| `claude -c` | ادامه آخرین مکالمه همین پروژه |
| `claude -c -p "query"` | ادامه آخرین مکالمه با یک دستور جدید |
| `claude -r` | انتخاب و ادامه مکالمه قبلی |
| `claude update` | آپدیت Claude Code |
| `claude --help` | دیدن راهنمای CLI |
