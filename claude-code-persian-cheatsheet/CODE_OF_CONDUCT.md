# آیین‌نامه رفتاری

هدف این پروژه ساخت یک مرجع فارسی مفید و محترمانه برای کاربران Claude و Claude Code است.

## رفتار قابل قبول

- احترام به دیگران
- نقد دقیق و بدون توهین
- اصلاح مستندات با منبع
- پذیرش اینکه دستورهای Claude Code ممکن است با نسخه‌ها تغییر کنند

## رفتار غیرقابل قبول

- توهین، تحقیر یا حمله شخصی
- انتشار اطلاعات خصوصی
- ارسال کلید API، توکن، رمزعبور یا داده محرمانه
- اضافه‌کردن محتوای گمراه‌کننده بدون هشدار یا منبع
