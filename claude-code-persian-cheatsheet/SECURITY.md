# سیاست امنیتی

این ریپوزیتوری آموزشی است و نباید شامل اطلاعات محرمانه باشد.

## قبل از انتشار عمومی بررسی کن

```bash
grep -R "API_KEY\|SECRET\|TOKEN\|PASSWORD" .
```

## گزارش مشکل امنیتی

اگر داخل پروژه اطلاعات حساس، لینک خصوصی، توکن یا راهنمای ناامن دیدی، issue عمومی با جزئیات محرمانه نساز. ابتدا مشکل را به شکل کلی گزارش کن و جزئیات حساس را حذف کن.
