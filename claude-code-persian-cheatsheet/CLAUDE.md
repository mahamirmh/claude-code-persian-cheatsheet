# دستورالعمل پروژه برای Claude Code

این پروژه یک ریپوزیتوری مستنداتی فارسی درباره Claude و Claude Code است.

## سبک نوشتار

- زبان اصلی: فارسی
- لحن: مستقیم، کاربردی، بدون اغراق
- ساختار: تیترهای واضح، جدول‌های کوتاه، مثال‌های قابل کپی
- از ادعای قطعی درباره دستورهای ناپایدار پرهیز کن
- برای دستورهای Claude Code، اگر مطمئن نیستی بنویس «اگر در نسخه شما فعال باشد»

## قوانین ویرایش

- اطلاعات خصوصی، API Key، Token یا رمزعبور اضافه نکن
- لینک منابع رسمی را حفظ کن
- README را خلاصه و کاربردی نگه دار
- توضیحات بلندتر را داخل پوشه `docs/` قرار بده
- پرامپت‌های آماده را داخل `examples/prompts/` بگذار
